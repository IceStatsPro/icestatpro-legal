🆓 Free Plan

The essentials to get you started — track fish appearances and see what's coming next. No payment required.

Getting Started
  1. 1Open Telegram → search @IceStatProBot
  2. 2Tap /start → tap Open IceStat Pro
  3. 3The app loads — you are on the Free plan by default
  4. 4Tap the ⏱ Session tab → enter a name (optional) → tap ▶ Start Session
Features Available

🐟 Next Fish Card

Shows which fish colour is estimated to appear next and at what time. Refreshes automatically every minute.

🔴 🟠 🔵 Fish Log Buttons

Three colour buttons on the main screen. Tap the matching colour the moment you spot a fish. Logging is only active during a session.

📋 Personal Log View

Scroll down to see your recent logged entries. Each entry shows the fish colour and exact time it was logged. Tap ✕ on any entry to remove it if logged by mistake.

⏱ Session Tracking

Start and end sessions to organise your fishing activity. All logs must be inside an active session to count.

How to Log a Fish
  1. 1Make sure a session is active (green indicator in Session tab)
  2. 2When you spot a fish, tap its colour button immediately
  3. 3The entry is saved with the current time
  4. 4The Next Fish card updates to reflect your new log
Tip: Log the fish the moment you see it — even a 30-second delay affects timing accuracy.
If you log by mistake, scroll down in the log and tap the ✕ button to remove it before ending the session.
Ending a Session
  1. 1Tap the ⏱ Session tab
  2. 2Tap ⏹ End Session
  3. 3Your session data is saved locally
Upgrading Your Plan

Additional Features

📡 Activity Feed (Queue)

Shows all 3 fish ranked by next expected appearance. Each card displays the estimated time, window range, and pattern strength score. The most imminent fish sits at the top.

🔥 Heatmap Tab

A minute-by-minute frequency map from :00 to :59. Brighter/coloured cells mean fish appeared more often at that minute historically. Outlined cells show currently active hot minute slots. Scroll left/right to see the full 60-minute view.

⏱ Session History Tab

Full list of all your past sessions with start time, end time and total duration. Helps you identify your most productive fishing windows.

📤 Export Data

Tap Export in the Log tab to download your full fish log as a CSV file. Useful for personal record keeping or sharing with others.

📥 Import Data

Tap Import to restore a previously exported CSV file back into the app. Useful if you reset the app or switch devices.

🔬 Analyze Data

Tap the ANALYZE DATA button in the Log tab. It syncs your entries to the central database for admin review and shows your personal stats:

• Total fish logged (all colours)
• Per fish: count + peak activity hour

How to Read the Heatmap
  1. 1Tap the 🔥 Heat tab
  2. 2The map scrolls to the current minute automatically
  3. 3Coloured cells = historically active minutes for that fish
  4. 4Bordered cells = currently active hot windows
  5. 5White outline = current minute · Yellow outline = next minute
Tip: Watch for clusters of coloured cells — a cluster of 3–4 consecutive minutes means fish appear reliably in that window.
Submitting Data for Approval
  1. 1After a session, tap 🔬 ANALYZE DATA
  2. 2Your entries are sent to the admin queue for review
  3. 3Once approved, they improve the central analysis model
Thank you: Your approved entries improve accuracy for everyone.
Managing Your Plan
  1. 1Tap 🔄 Manage Plan in the top right corner
  2. 2Your current plan and expiry are shown at the top
  3. 3Switch to Premium tab to upgrade
  4. 4Renew by selecting the same or longer duration

👑 Premium Plan

The complete IceStat Pro experience. Real-time burst detection, state badges, in-app alerts and background Watch Mode notifications.

✅ Includes everything in Free and Advanced
Exclusive Premium Features

⚡ Burst Mode Banner

A live banner at the top of the screen shows the current fish activity state:
⚡ BURST MODE — fish are appearing rapidly, stay alert and log every sighting.
💤 REST MODE — fish activity is low, relax between sessions.

🏷️ State Badges in Queue

Each fish card in the Activity Feed shows its real-time state:
⚡ BURST — this fish is in an active burst cycle, expect rapid appearances.
💤 REST — this fish is resting, next appearance is further away.

📊 Two-State Hint

Below the estimated time on the hero card, you'll see both state estimates simultaneously:
⚡ Burst ~4m — expected gap if currently in burst mode
💤 Rest ~25m — expected gap if currently in rest mode

🔔 In-App Burst Alert

When any fish transitions from Rest → Burst, the app automatically:
• Vibrates your phone
• Plays a 3-tone alert sound
• Shows a green animated banner

This tells you to start watching immediately. There is a 5-minute cooldown per fish to avoid repeated alerts.

👁️ Watch Mode (Background Notifications)

Get Telegram notifications even when the app is closed:

Setting Up Watch Mode
  1. 1Tap the 🔔 button in the top bar
  2. 2Watch Mode activates — the button turns green
  3. 3Close IceStat Pro and go about your day
  4. 4When a fish enters Burst Mode, you receive a Telegram message instantly
  5. 5Tap the notification to open IceStat Pro directly
Best practice: Enable Watch Mode during long breaks between fishing sessions so you never miss a burst window.
Battery note: Watch Mode uses Telegram's own notification system — it does not drain your battery like a background app.
Tips for Premium Users
During Burst Mode: Log every fish you see — the more data during bursts, the more accurate future burst detection improve.
Rest periods: When 💤 REST is shown, use the two-state hint to decide whether to keep watching or take a break.
Watch Mode + Burst: Enable Watch Mode before stepping away. You'll be back exactly when burst begins.
Model switching: As an Advanced/Premium user, you can see the admin switch between analysis models (V2 and V3). V3 is the recommended default — it uses minute-pattern analysis based on all historical data.
